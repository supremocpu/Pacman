<p align="center">*PACMAN GAME USING JFRAME*</p>

A simple Pac-Man game implemented in Java using JFrame for the GUI.  
Followed a tutorial by Kenny Yip Coding https://www.youtube.com/watch?v=lB_J-VNMVpE

---

## Features

- Classic Pac-Man gameplay  
- Graphical interface using Java JFrame  
- Movable Pac-Man character with keyboard controls  
- Collectibles and basic game logic  

---

1. Run the program in a Java IDE or from the command line.  
2. Use the arrow keys to move Pac-Man around the maze.
